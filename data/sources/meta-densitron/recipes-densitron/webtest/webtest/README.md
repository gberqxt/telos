# Test app for Telos project.

This app include:
    - Rotary switch test
    - Rotary switch button test
    - Led Test
    - Audio Test
    - Speaker volume (left and right)
    - Headphone volume (Left and Right)
    - Microphones test
    - Front Microphone gain controls
    - Headphone Microphone gain controls
    - simple audio files managment


## How To:

You need to install flash:
    pip3 install -r requirements.txt

Navigate to program folder,

run with:
    python3 ./app.py
